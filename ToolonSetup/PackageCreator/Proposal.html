Make pkg project

First. You have to make pkg project directory.
Follow the way in below.

Tools on Developper shell :
>pkgcreate new (Package name)

Then project will be make at your current directory

Config install setting

Next. You have to edit config file. Don't worry. It's so Easy.

There are three configuretion file in pkg project directory

  • ilcfg.pkgcfg (Application Setting system Configuration)
  • pkgcfg.pkgcfg (Application Installing location Configuration)
  • pkgname.pkgcfg (Application Information Configuration)

First. You have to write pkgname.pkgcfg like below example

pkgname.pkgcfg (Example) :
name:(Your application name - Don't Use ' : ' in value)
version:(Your application version - Don't Use ' : ' in value)

Secound. You have to write ilcfg.pkgcfg like below example

ilcfg.pkgcfg (Example) :
gcfile:(Global Config File name)
ncfile:(Naitive Config File name)

Third. You have to write pkgcfg.pkgcfg like below example

pkgcfg.pkgcfg (Example) :
(File name in "bin" Directory):(Filename of put to "Application dir\Your application's name")
~ each file in bin directory ~
