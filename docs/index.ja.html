Tools on

Toolson は、非常に便利なマルチツールです

「Tools」って何？

Toolsは道具と同じ意味です。
これらの道具は、あなたのPCやスマートフォンライフをより良いものにするでしょう。

使い方

まず、Tools on Frame workをインストールしてください。（只今、インストールしなくても使えるバージョンを開発しております）
そうすれば、殆どのアプリケーションを使うことができます。 (実行ファイル及びソースコードは ダウンロードページ もしくは 公式Github よりダウンロードできます)

インストーラーも開発中です

安全なの？

安全です
もし、信用できないなら、公開されているコードを見て、自分でコンパイルしてみてください。手順書を準備しておきますので

しかし、当たり前ですが、Toolsonのブラウザを使った、外部サイトでのウイルスなどへの感染などは防げません。
このことをご了承ください

対応端末について

当アプリケーションは、Windowsと一部のAndroidを含むLinuxに対応しています
ただ、アプリケーションによっては正常に動かない場合があります

対応表をご用意いたしましたので、ご覧ください
