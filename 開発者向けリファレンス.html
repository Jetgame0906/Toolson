はじめに

この度はTools onをご利用いただきましてありがとうございます。
当アプリケーションは、サードパーティーのアプリケーションが開発しやすいように多数のAPIや言語をサポートしております。
当ドキュメントでは、それらのAPIや使用可能言語をご紹介いたします。

Tools on の仕組み

Tools onは、Windowsオペレーティングシステム上で動作するアプリケーションで、ユーザーの理想のPCライフの手助けをモットーに開発されております。
当アプリケーションは、C#をはじめとする様々な言語で構築されておりますが、サードパーティー向けの言語としてRubyとPHPをサポートしております。

Tools onは、一部アプリケーションやサービスを除いて独立型のアプリケーション（Tools on Frameは必須）となっており、それらのプログラムは、ユーザーが自由に選択できます。

Tools on Scriptについて

Tools on には、先ほど申しましたとおり、サードパーティー向け言語のサポートがついております。
これらは、無料で使用することができ、ユーザーの理想をかなえるお手伝いをいたします。
現状、使用できる言語は、以下の通りです。

  • Ruby
  • PHP

また、それぞれの言語にはAPIという名の関数セットが用意してあり、簡単にTools onの機能を利用することができます。

Tools on 上でのWindowsプログラム

Tools onはWindowsプログラムですので、Windows上で、動作ができるプログラムは問題なく使用できます。
こちらも、スクリプトサポートと同じく、バイナリーファイルとして、Tools onの機能をご利用いただけます

Tools on Binary API について

こちらのAPIは開発段階です

Tools on APIには、様々な機能がありますが、現在使用できるものは、ごくわずかです。
今回は、その一部をご紹介します。
